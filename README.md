# GameTrack 🎮

Sistema de gestión de tickets para emprendimiento de juegos infantiles.

## Stack
- **Backend:** FastAPI + PostgreSQL
- **ML:** scikit-learn + MLflow
- **GenAI:** LangChain + Claude API
- **Infra:** Docker + GitHub Actions

## Módulos
- Capa 1: App operativa (tickets en tiempo real)
- Capa 2: Analytics y predicción de demanda
- Capa 3: IA Generativa sobre data propia
- Capa 4: MLOps con reentrenamiento automático
